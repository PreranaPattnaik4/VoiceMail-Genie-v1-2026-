# VoiceMail-Genie App Screenshots

This directory is designated for high-resolution visual screenshots of the VoiceMail-Genie v1 responsive application dashboard.

## Suggested Screenshots to Capture
1. **`01_authentication_gate.png`:** The sleek indigo dark-theme sign-in screen, displaying both Email/Password inputs, Google federated controls, and the instant sandbox-mode toggle.
2. **`02_main_dashboard.png`:** The fully-loaded Bento Grid interface containing the live audio visualizer, scenario preset selectors, active composer editor, and performance analytics cards.
3. **`03_active_recording.png`:** The visual microphone card during an active recording state, showing the glowing animated waveform and counting timer.
4. **`04_draft_generation.png`:** The generated email draft inside the rich text composer, highlighting subject alignments and actionable bulleted takeaways.
5. **`05_gmail_sync_success.png`:** The workspace card showing the "AUTHORIZED" checkmark and the success toast banner confirming direct mailbox insertion.

## Capture Guidelines
* **Aspect Ratio:** Capture in standard modern desktop viewports (e.g., `1920x1080` or `16:9` ratio) and mobile vertical mockups (`9:16`).
* **Format:** Save files as high-quality, lightweight PNGs to ensure clarity in GitHub Markdown documentation.
